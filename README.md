# deeplay-test
Test task for Deeplay Intership.

Проект представляет собой консольное приложение.
Данные между запусками не сохраняются, в ключевой класс можно добавить взаимодействие с базой данных. Есть куда развиваться.

# Tools
- TypeScript v4.7.3
- Node JS v16.13.2

# Usage

## Before start
```
npm install
```

## Tests
```
npm run test
```

## Run
```bash
tsc
node dist/app.js
```
